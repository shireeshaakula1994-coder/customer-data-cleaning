# 📊 Customer Data Cleaning & Preprocessing

## 📌 Project Overview

This project focuses on cleaning and preprocessing a customer dataset to
make it ready for machine learning models.

------------------------------------------------------------------------

## ✅ Tasks Performed

### 1. Data Cleaning

-   Imputed missing values:
    -   Age → filled using median
    -   City → filled using mode
-   Removed duplicate rows

### 2. Encoding Categorical Variables

-   City → One-Hot Encoding
-   Gender → Label Encoding (Male = 1, Female = 0)

### 3. Feature Scaling

-   Applied Min-Max Scaling
-   Applied Standardization (Z-score normalization)

------------------------------------------------------------------------
